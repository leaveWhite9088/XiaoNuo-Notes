## 第二部分：数字故障代码详解 (Error Codes 100-199)

### 100-199：传感器与导航系统故障

#### Error 101 - 激光雷达 (LDS) 测距异常
* **故障描述**：机器启动后，顶部的激光塔转动几圈后停止，随后报错。
* **技术原理**：激光头无法旋转（皮带打滑/电机损坏）或激光接收二极管无信号。
* **详细排查步骤**：
    1.  **目视检查**：透过激光罩缝隙，观察橙色皮带是否断裂或脱落。
    2.  **物理拨动**：用棉签轻轻拨动激光头，检查是否有强烈阻尼感（电机抱死）或毫无阻力（皮带断裂）。
    3.  **异物清理**：检查激光发射/接收窗口（两只“眼睛”）是否被柳絮、蜘蛛网遮挡。

#### Error 102 - 碰撞缓冲器 (Bumper) 卡死
* **故障描述**：机器在空旷处后退、转圈，或提示“请敲击保险杠”。
* **排查步骤**：
    1.  检查机身前侧半圆弧撞板的缝隙中是否卡入了乐高积木、瓜子壳等硬物。
    2.  双手反复按压撞板左、中、右三个位置，正常情况下应有清脆的“咔哒”回弹声。若无回弹，说明微动开关卡死。

#### Error 103 - 悬崖传感器 (Cliff Sensor) 触发
* **故障描述**：机器在平地（非楼梯口）突然急刹车并报错，或原地转圈后报错。
* **特殊场景分析**：
    * **场景 A（脏污）**：翻转机器，用干布擦拭底部 6 个黑色长条状传感器窗口。
    * **场景 B（误判）**：机器处于深色/黑色地毯上，或强光直射地面。深色材质会吸收红外光，导致传感器认为机器已悬空。**解决方案**：在 APP 中关闭“防跌落增强”模式（需谨慎）。

#### Error 104 - 沿墙传感器 (Wall Sensor) 异常
* **故障描述**：机器无法直线贴墙，呈波浪形行走，或距离墙面过远 (>3cm)。
* **排查**：检查机身右侧的小黑窗是否沾染了油污或灰尘。

#### Error 105 - 惯性导航单元 (IMU) 数据漂移
* **故障描述**：地图中的路径显示机器穿墙而过，或原地旋转无法定位。
* **校准流程**：
    1.  将机器关机。
    2.  放置在绝对水平的硬质地面上（严禁在地毯上）。
    3.  长按“回充键”+“电源键”10秒，直到听到“开始校准IMU”提示音。
    4.  静置 5 分钟不动，直到听到校准完成提示。

#### Error 109 - 3D 结构光模组异常
* **故障描述**：机器直冲撞向拖鞋、线缆，避障功能失效。
* **排查**：检查机身正前方的玻璃面板是否有严重划痕或油污。

## 第三部分：驱动与机械故障 (Error Codes 200-299)

#### Error 201 - 主轮悬空 (Wheel Floating)
* **原因**：机器爬上了门槛条、电风扇底座，导致驱动轮离地，微动开关被释放。
* **解决**：将机器搬回平地，按清扫键继续。

#### Error 202 - 主轮过载 (Wheel Overload)
* **故障描述**：机器尝试移动但无法前进，伴随电机“嗡嗡”堵转声。
* **排查步骤**：
    1.  翻转机器，用力按压主轮，检查能否正常回弹。
    2.  转动轮子，检查轮边缝隙是否缠绕了大量头发。
    3.  检查是否有小石子卡在轮胎花纹与机壳之间。

#### Error 203/213 - 边刷电机故障 (Side Brush)
* **Error 203 (左边刷) / Error 213 (右边刷)**
* **原因**：边刷被地毯流苏、数据线死死缠绕。
* **解决**：必须使用十字螺丝刀拆下边刷螺丝，取下塑料刷体，清理隐藏在根部转轴上的头发（此处最容易被忽略）。

#### Error 204 - 主刷 (Roller Brush) 缠绕
* **原因**：吸入了袜子、整团纸巾、抹布。
* **解决**：取出主刷，取出两端的可拆卸端盖，清理轴承。

#### Error 205 - 拖布支架升降机构卡滞
* **故障描述**：从地毯移动到地板时，拖布无法降下，或回充时无法抬起。
* **排查**：
    1.  检查拖布支架与机身连接处是否有积尘结块。
    2.  手动用力按压支架几次，辅助电机复位。

## 第四部分：水路与风路系统故障 (Error Codes 300-399)

#### Error 301 - 尘盒未安装或霍尔感应失效
* **故障描述**：尘盒明明装进去了，机器还是报“请安装尘盒”。
* **深度技术排查**：
    1.  **磁铁丢失**：检查尘盒左侧的小凹槽内，是否有一块银色的小磁铁脱落。如果是，需要更换尘盒。
    2.  **感应失效**：用一块强磁铁靠近机身尘盒仓左侧的感应点，若能消除报警，说明是尘盒磁力不足。

#### Error 302 - 风机电流异常 (Fan Motor Error)
* **高危警告**：此错误通常意味着风机吸入了水。
* **紧急处理**：
    1.  立即关机，取出尘盒。
    2.  检查机身吸尘口内部是否有水珠。
    3.  **必须静置晾干 48 小时以上**。切勿尝试开机，否则会导致主板烧毁。

#### Error 310 - 清水箱“假性”缺水
* **故障描述**：基站清水箱是满的，但基站一直提示“请加清水”。
* **原因**：清水箱内的水位浮漂被气泡吸附或卡住。
* **解决**：
    1.  打开清水箱盖，用手指上下拨动一下角落里的浮漂，确认其能浮起。
    2.  检查基站水箱槽底部的密封圈是否破损。

#### Error 312 - 基站不出水/水泵气锁 (Air Lock)
* **故障描述**：清洗拖布时，清洗盘是干的，水泵有声音但不出水。
* **专业处理方案 (气锁排除法)**：
    1.  这是因为新机或长期未用的机器，水管内充满了空气，导致水泵空转。
    2.  取下清水箱，在基站入水口处滴入几滴水。
    3.  找一个**注射器**，对准基站入水口，用力推入 20ml 清水，强制排出管道空气。
    4.  重新安装水箱，执行一次“自清洁”。

## 第五部分：电池与充电故障 (Error Codes 400-499)

#### Error 401 - 红外回充信号丢失
* **故障描述**：机器在基站面前反复转圈，就是对不准，最后报错。
* **排查**：
    1.  检查基站正面的黑色透光条是否被贴纸遮挡或脏污。
    2.  检查是否有阳光直射基站（阳光中的红外线会干扰信号）。

#### Error 402 - 充电接触不良
* **解决**：用酒精棉片用力擦拭机器尾部的两个金属极片，以及基站内部的弹簧顶针。这通常是由于金属氧化层导致的。

#### Error 404 - 电池管理系统 (BMS) 通讯失败
* **原因**：电池接口松动或非原厂电池。
* **解决**：若为新机，需返厂维修。若为老机器，请检查电池插头是否氧化。

## 第六部分：非代码类疑难杂症排查 (Expert FAQ)

### Q1: 机器人频繁离线，如何进行深度网络诊断？
请按以下**决策树**进行排查：
1.  **Step 1: 检查路由器频段**
    * 是 5G Wi-Fi 吗？ -> **是**：不支持，请连接 2.4G 网络。
    * 是 2.4G/5G 双频合一吗？ -> **是**：部分路由器兼容性差，请在路由器后台将 2.4G 和 5G 分开命名（例如：MyHome_2.4G）。
2.  **Step 2: 检查 SSID 和密码**
    * 包含中文、空格、特殊符号（@#¥%）吗？ -> **是**：请修改为纯英文字母+数字。
3.  **Step 3: 检查网络协议**
    * 是 WPA3 加密吗？ -> **是**：请降级为 WPA2-PSK 混合模式。
    * 是企业级认证（需要网页登陆）吗？ -> **是**：不支持。

### Q2: 拖地后地面有明显水渍、轮胎印或异味？
**综合解决方案矩阵**：
* **异味问题**：
    * *成因*：滤网没晾干就安装 = 发霉味；污水箱没洗净 = 腐烂味。
    * *对策*：滤网水洗后必须暴晒 24h；污水箱需用试管刷刷洗内壁。
* **水渍/轮胎印**：
    * *成因*：轮子沾了油污；基站清洗拖布不彻底。
    * *对策*：用湿巾擦拭两个驱动轮；在 APP 调高“回洗频率”（设为 10 分钟一次）。

### Q3: 为什么最近地图总是乱了/叠图？
**技术分析**：
SLAM 算法依赖特征点匹配。地图重叠通常是因为环境特征发生剧变。
1.  **基站位移**：您是否移动了基站？哪怕移动 10cm，也可能导致机器出发时坐标系错误。
    * *解决*：将机器放回基站，让它自己从基站出发。
2.  **环境相似度**：您是否在完全对称的长走廊使用？
    * *解决*：在走廊贴一些贴纸作为参照物。
3.  **恢复方法**：不要删除地图重来！请进入 APP -> 地图编辑 -> **历史地图**，找回 3 天前那个正常的版本点击“应用”。
