## 使用说明

- **第1合集**：大模型入门与实践系列教程，涵盖 LoRA 微调、本地知识库搭建等内容
- **第2合集**：大模型理论加实测系列，包含各大模型的能力测评（MiniMax M2.5/M2.7/M3、美团 LongCat 2.0）以及 Token 退化问题深度分析（马嘉祺现象、迷失传奇诊断）
- **第3合集**：开发工具评测与模型速率横评，涵盖低价接入方案、模型横评、CodingPlan 速率对比

每个文件夹对应一个视频，点击标题即可跳转到对应章节。资料包含教程文档、源码、演示截图、论文资料等。

如有疑问可前往 B 站视频评论区留言。
