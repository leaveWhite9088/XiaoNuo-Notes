## 测试亮点

- **前端落地页**：验证 LongCat 2.0 对品牌视觉、响应式布局与交互动效的代码生成能力。
- **办公搜索**：结合公开政策库、官方文档与 PDF 材料进行交叉验证，测试信息检索与合规整理能力。
- **后端迁移**：对比 Python/Django 大版本升级，评估模型在代码现代化、废弃 API 修复与插件系统设计上的能力。
