## 趋势分析

### 上半年定价关键趋势

1. **价格战分化**：海外三家维持"稳价提性能"路线（Opus 从 $15→$5 是一次性调整，之后价格不变模型迭代）；国产则持续"降价 + 降价永久化"（DeepSeek、小米）

2. **隐性涨价成为常态**：
   - Anthropic Opus 4.7 tokenizer +35% token 膨胀 → 实际单次请求涨 35-46%
   - OpenAI GPT-5.5 输出价 $15→$30 翻倍
   - Google AI Pro 静默限额收紧
   - 阿里云算力基础设施涨价 34%
   - 智谱 3 倍高峰消耗 = 标价的 1/3 实际额度
   - MiniMax 从 Coding Plan 改 Token Plan + 全模态共享池

3. **5 小时滚动窗口成为行业标准**：Anthropic、Google、智谱、Kimi 均采用 5h 刷新 + 周限额双重门控

4. **Coding Plan 大战**：智谱/阿里/火山方舟/Kimi 四家 Coding Plan 直接对标 Claude Code。起步价 ¥40-49/月，远低于 Claude Pro $20

5. **缓存定价成为杀手锏**：DeepSeek V4-Pro 缓存命中仅 $0.003625/M（是 Claude Opus 的 1/138）；Kimi K2.6 缓存命中 $0.07/M

6. **Agent 时代推高实际成本**：智谱用户实测单 Agent 1 小时烧 1.2 亿 token。Agent 循环 + 工具调用使理论低价在实践中爆炸

### 国内 vs 海外价差

以旗舰模型为例（$/M tokens 输出）：
- Claude Opus 4.8: $25 → DeepSeek V4-Pro: $0.87 = **28.7 倍差价**
- GPT-5.5: $30 → Qwen3.5-Flash: $0.26 = **115 倍差价**
- 即便对标同级（Opus vs GLM-5.2）: $25 vs $3.20 = **7.8 倍差价**
