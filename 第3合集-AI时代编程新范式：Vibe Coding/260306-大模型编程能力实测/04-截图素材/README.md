# 截图素材说明

本目录把第一期 Bilibili One-Shot 横测截图单独整理成视频素材库。

## 结构

- `按模型/`
  - 适合逐个模型讲解，保留每个模型的完整截图槽位。
- `按功能位/`
  - 适合横向对比同一功能：
  - `A-首页主界面`
  - `B-视频播放与信息展示`
  - `C-评论与弹幕区域`
  - `D-登录入口或登录页`
  - `E-收藏或个人中心`
  - `F-目录结构与架构总览`

## 说明

- PNG 为主要讲解截图。
- SVG 主要用于目录结构与失败态/架构总览占位。
- 原始网页报告中的截图仍保留在 `01-报告网页/screenshots/`，这里是额外整理后的便捷取图目录。
