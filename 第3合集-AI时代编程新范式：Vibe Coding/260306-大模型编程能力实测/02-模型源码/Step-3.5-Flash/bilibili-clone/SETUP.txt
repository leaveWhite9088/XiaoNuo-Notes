Bilibili 克隆项目 - 设置说明
=================================

项目结构：
- backend/   - Node.js + Express 后端 API 服务
- frontend/   - React + TypeScript + Tailwind CSS 前端应用

前置要求：
- Node.js >= 18
- npm >= 9

快速开始：

1. 安装依赖：
   cd backend && npm install
   cd ../frontend && npm install

2. 配置环境变量：
   cp backend/.env.example backend/.env
   编辑 backend/.env 根据需要修改配置

3. 启动开发服务器：
   方式一：使用提供的 run.sh 脚本（Mac/Linux）
      chmod +x run.sh
      ./run.sh

   方式二：手动启动
     终端1: cd backend && npm run dev
     终端2: cd frontend && npm run dev

4. 访问应用：
   前端：http://localhost:5173
   后端 API：http://localhost:3001/api

功能说明：
- 浏览热门视频、视频搜索
- 视频播放（支持多清晰度）
- 实时弹幕覆盖显示
- 评论浏览与发布
- 用户登录/登出
- 查看个人收藏夹（需要登录）
- 发送弹幕和评论（需要登录）
- 跳转B站官网

技术栈：
后端：
  - Express.js
  - bilibili-api (Bilibili API 封装)
  - express-session (会话管理)
  - CORS, Helmet 等安全中间件

前端：
  - React 18
  - React Router v6
  - Tailwind CSS (Bilibili 风格配色)
  - Video.js (视频播放器)
  - Axios (HTTP 请求)

注意：
- 本项目使用 bilibili-api 库直接调用 Bilibili 官方 API
- 需要有效的 Bilibili 账号密码才能登录并使用需认证功能
- 仅用于学习和研究目的

遇到问题：
- 确保后端 .env 已正确配置
- 检查端口 3001 和 5173 未被占用
- 查看浏览器控制台和后端终端日志
