# README

本期资料对应“大模型一轮直出搭建 Bilibili 视频平台”横向测评。

## 包含内容

- 最终对外展示网页
- 所有模型统一截图槽位
- 各模型源码（不含依赖）
- 报告生成脚本与评分覆写脚本

## 不包含内容

- `node_modules`、虚拟环境、构建缓存
- 登录凭据、Cookie、二维码登录中间文件
- 本地数据库、日志、临时调试产物

## 使用建议

- 直接打开 `01-报告网页/index.html` 可本地浏览最终报告。
- 若需要看源码对比，从 `02-模型源码/` 按模型目录进入。
- 若需要重生成网页报告，优先查看 `03-评测工具/`。
